## Instalación Local

### Requisitos
- Node.js (versión 16+)

### Pasos
1. Abrir terminal en la carpeta del proyecto
2. Ejecutar:
```bash
npm install
npm run start
```
3. La aplicación se abrirá en: `http://localhost:9002`
